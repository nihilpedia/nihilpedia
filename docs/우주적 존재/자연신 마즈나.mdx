# 개요
<QBlock>
  <P><Strong>이 세상의 자연들이 그대들에게 힘을 내어줄 것이다.</Strong></P>

[질서의 기운](질서의_기운)을 이루는 [절대신](절대신)중 하나. 자연을 상징한다. [원소의 요람](마즈나_엘레멘티아)에서 기거하며 [세계수](세계수)에 양분을 공급하는 꽃과 식물을 보살피는 역할을 한다.

# 상세
자연신 마즈나는 본래 [질서신 니힐라](질서신_니힐라)와 [혼돈신 디모니아](혼돈신_디모니아)의 전쟁에 관심이 없었다. 누가 이겨도 아무런 상관 없는 결과물이었을테니. 하지만 질서신 니힐라는 승리를 취하자마자 질서의 평원을 개척했고 세계수의 씨앗을 심었다. 이를 보고 그에게 힘을 가세하기로 한다.

원소의 요람에는 한정적인 종류의 식물 밖에 없기 때문에 종종 세계수 안으로 들어가 신기하고 아름다운 식물을 가져오기도 한다. 자연신 마즈나는 여성의 외형을 고집한다고 하며, [엘레멘티](엘레멘티)라는 **정령**들을 만든 존재이기도 하다.

엘레멘티는 필멸자들과 계약하여 자연의 힘인 마나를 사용할 수 있게 해주는 역할을 갖게 된다.
