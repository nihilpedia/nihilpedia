# 개요
니힐라 위키는 필자인 <A href='https://www.instagram.com/nihil_illust/' type='external'><Strong>니힐</Strong></A>이 소설을 집필하면서 만든 창작 세계관인 [**니힐라 엔시유니아**](니힐라_엔시유니아)의 설정 문서를 기록하고 보관하기 위해 만들어졌다. 니힐라 위키 자체에 대한 자세한 사항은 [소개](소개)페이지를 참조하기 바란다.
